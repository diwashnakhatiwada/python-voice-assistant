# Python Voice Assistant 🎙️

A smart voice-controlled assistant built with Python that can understand voice commands and perform everyday tasks. The assistant can greet users, answer questions, tell the current date and time, introduce the developer, open applications like the camera, launch websites such as Google and YouTube, and play music using simple voice commands.

## ✨ Features

* 🎤 Voice command recognition
* 🗣️ Text-to-speech responses
* 📅 Tells the current date and time
* 📷 Opens the system camera
* 🌐 Opens Google and YouTube
* 🎵 Plays music
* 👩 Personal introduction on voice command
* 🚪 Exit command

## 🛠️ Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* OpenCV
* PyAudio
* Webbrowser
* Playsound

## 📌 Purpose

This project demonstrates how voice recognition and automation can be combined to build an interactive virtual assistant. It can also be customized for workplaces to help new employees by answering common questions, providing project information, and improving productivity.
